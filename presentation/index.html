Научная проблема

Почему это сложно?

  • Неравномерность данных: анализ значимости сосредоточен на 1973–2026 (2041 событие); каталог NOAA (исторические + инструментальные записи) с кардинально разной плотностью записей
  • Проблема полноты: в древних записях зафиксированы только катастрофические события; мелкие землетрясения систематически пропущены
  • Пространственная неоднородность: плотность сейсмических станций сильно варьируется по регионам

Проблема полноты каталога

Закон Гутенберга–Рихтера

\[ \log_{10} N = a - b \cdot M \]

N — число событий с магнитудой ≥ M  ·  b ≈ 1.0 (глобальная константа)  ·  Mc — магнитуда полноты

Период           Mc (глобально)   Событий/год
─────────────────────────────────────────────
2000–2026             6.0           ~3 000
1950–1999             6.5           ~800
1900–1949             7.0           ~300
1800–1899             7.5           ~80
до 1800               8.0           ~15

Решение: completeness_weight — взвешивание каждого события по обратной вероятности его обнаружения в данном регионе и эпохе.

Эвристическая метрика с тектонической подсказкой

Проблема

Обычное евклидово расстояние игнорирует геологическую структуру Земли.

Аналогия: расстояние «напрямую через горы» vs. путь по реальному шоссе вдоль долин.

Решение: Bird (2003)

Граф 20 сегментов границ литосферных плит.

Алгоритм Дейкстры для кратчайшего пути.

Событие A ──→ граница A'
                  │
            Дейкстра: кратчайший путь по графу разломов
                  │
              граница B' ──→ Событие B

r_tect(A,B) = dist(A, A') + path(A', B') + dist(B', B)

Метрика η (Baiesi–Paczuski)

\[ \Large \eta_{ij} = t_{ij} \cdot r_{ij}^{\,1.6} \cdot 10^{-m_i} \]
Член формулыФизический смысл
\( t_{ij} \) «Насколько давно произошёл предыдущий толчок»
\( r_{ij}^{1.6} \) «Как далеко (по разломам) он произошёл»
\( 10^{-m_i} \) «Насколько мощным он был» (коррекция на магнитуду)

Малый η

→ события связаны
(близко, недавно, мощное)

Большой η

→ события случайны,
независимы

Источник: Baiesi & Paczuski (2004) PRE — адаптировано с тектоническим расстоянием

Алгоритм поиска глобальных серий

for j in events:
    parent[j] = argmin_i η(i → j)  # ближайший «родитель»

# Построить дерево связей (nearest-neighbour graph)

for T in [1day, 1week, 1month, 1year, 10years]:
    window = events in T with M ≥ 6.5
    if (len(window) >= 4
        and unique_regions(window) >= 3
        and eta_connectivity(window) > threshold):
        → кандидат на глобальную серию

for candidate in candidates:
    p_value = monte_carlo_test(candidate, N=10_000)
    if p_value < 0.01:
        → ✓ значимая глобальная серия

Ключевой параметр: порог η разделяет «афтершоки» от «независимых» событий по бимодальному распределению (Zaliapin & Ben-Zion, 2013).

Статистическая проверка

Нулевая гипотеза (H₀)

Сейсмические события образуют пуассоновский процесс — независимы, равномерно распределены во времени при фиксированной пространственной структуре.

Монте-Карло тест (N = 10 000)

  1. Зафиксировать координаты и магнитуды
  2. Случайно переставить временны́е метки
  3. Вычислить тестовую статистику кластеризации
  4. p-value = доля симуляций с более сильным кластером
Наблюдаемая статистика: ──────────────────┤●
Распределение H₀:       ████▓▓▒▒░░░░░░░░░│   (хвост < 1%)
                                          ↑
                                       p < 0.01  ✓

Ограничения метода

  • Неполнота исторических данных
    Систематическая недооценка числа событий в слабо задокументированных регионах до ~1900 CE (Центральная Азия, Африка, Тихий океан)
  • Корреляция ≠ причинность
    Обнаружение серий не доказывает физическую связь между событиями
  • Чувствительность к параметрам
    Показатель df = 1.6 и b = 1.0 требуют региональной калибровки; глобальные усреднения могут маскировать локальные аномалии
  • Неопределённость дат
    Для событий BCE погрешность составляет ±10–100 лет, что существенно влияет на tij и значение η
  • Статичная модель плит
    Bird (2003) не учитывает эволюцию тектоники на исторических масштабах

I
