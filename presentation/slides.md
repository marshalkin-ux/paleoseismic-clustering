# Научная проблема

## Почему это сложно?

- **Неравномерность данных:** каталог охватывает 4175 лет (2150 BCE → 2026 CE) — с кардинально разной плотностью
- **Проблема полноты:** в древних записях зафиксированы только катастрофические события; мелкие землетрясения систематически пропущены
- **Пространственная неоднородность:** плотность сейсмических станций сильно варьируется по регионам

## Предыдущие работы и наш вклад

- *Hill et al. (1993), Brodsky & Prejean (2005)* — анализ только инструментального периода (после 1900)
- **Наш вклад:**
  - 4000+ лет данных из трёх источников
  - Тектоническое расстояние вместо евклидова
  - Статистическая коррекция полноты каталога

# Проблема полноты каталога

## Закон Гутенберга–Рихтера

$$\log_{10} N = a - b \cdot M$$

- **N** — число землетрясений с магнитудой ≥ M
- **b ≈ 1.0** — наклон (глобальная константа)
- **Mc** — магнитуда полноты (меняется по эпохам и регионам)

## Как меняется Mc со временем

```
Период           Mc (глобально)   Доступных событий/год
─────────────────────────────────────────────────────
2000–2026             6.0               ~3 000
1950–1999             6.5               ~800
1900–1949             7.0               ~300
1800–1899             7.5               ~80
до 1800               8.0               ~15
```

**Наше решение:** `completeness_weight` — взвешивание каждого события по обратной вероятности его обнаружения в данном регионе и эпохе.

# Ключевая инновация №1: Тектоническое расстояние

## Проблема евклидова расстояния

Обычное «расстояние по прямой» игнорирует геологическую структуру Земли.

> **Аналогия:** расстояние через горы по прямой vs. путь по реальному шоссе вдоль долин. Два события в разных океанических бассейнах могут быть «близко» по евклидовой метрике, но разделены тысячами км разломов.

## Решение: граф границ плит

На основе **Bird (2003)** построен граф из **20 сегментов** границ литосферных плит с взвешенными рёбрами (длина дуги).

```
Событие A ──→ ближайшая граница A'
                    │
              Дейкстра: кратчайший путь по графу границ
                    │
              ближайшая граница B' ──→ Событие B

r_tect(A, B) = dist(A, A') + path(A', B') + dist(B', B)
```

**Эффект:** события на разных плитах «дальше», чем подсказывает геометрия.

# Ключевая инновация №2: Метрика η (эта)

## Формула

$$\large \eta_{ij} = t_{ij} \cdot r_{ij}^{1.6} \cdot 10^{-m_i}$$

## Интерпретация каждого члена

| Член | Физический смысл |
|---|---|
| **t_ij** | «Насколько давно произошёл предыдущий толчок» |
| **r_ij^1.6** | «Как далеко (по разломам) он произошёл» |
| **10^{−m_i}** | «Насколько мощным он был» (коррекция на магнитуду) |

- **Малый η** → события связаны (близко во времени, пространстве, при сильном предшественнике)
- **Большой η** → события случайны, независимы

> *Источник: Baiesi & Paczuski (2004), PRE — адаптировано с тектоническим расстоянием вместо евклидова*

---

# Алгоритм поиска глобальных серий

```
ДЛЯ каждого события j:
    найти i* = argmin_i η(i→j)   # ближайший «родитель»
    добавить ребро i* → j в дерево связей

ПОСТРОИТЬ дерево связей (nearest-neighbour graph)

ДЛЯ каждого временного окна T ∈ [1 день ... 10 лет]:
    найти все события в окне T с M ≥ 6.5
    ЕСЛИ:
        - ≥ 4 событий в окне
        - охват ≥ 3 различных тектонических регионов
        - суммарная η-связность выше порога
    ТОГДА: кандидат на глобальную серию

ДЛЯ каждого кандидата:
    Monte Carlo тест (N = 10 000 перестановок)
    ЕСЛИ p-value < 0.01:
        → значимая глобальная серия ✓
```

**Ключевой параметр:** порог η разделяет «афтершоки» от «независимых» событий по распределению (бимодальность, Zaliapin & Ben-Zion 2013).

# Статистическая проверка

## Нулевая гипотеза (H₀)

Сейсмические события образуют **пуассоновский процесс** — независимы, равномерно распределены по времени при фиксированной пространственной структуре.

## Монте-Карло тест

1. Зафиксировать пространственные координаты и магнитуды
2. **Случайно переставить** временны́е метки (10 000 раз)
3. Для каждой перестановки вычислить тестовую статистику кластеризации
4. **p-value** = доля симуляций, давших кластер *сильнее* наблюдаемого

```
Наблюдаемая статистика: ──────────────────┤●
Распределение H₀:       ████▓▓▒▒░░░░░░░░░│  (хвост < 1%)
                                          ↑
                                       p < 0.01
```

> При **p < 0.01** отвергаем нулевую гипотезу: наблюдаемая серия не объясняется случайностью.

# Возможные физические механизмы

## Три гипотезы для глобальной связности

**Статический триггер** *(Stein et al., 1992)*
Изменение напряжения Кулона (ΔCFS ≈ ±0.1 бар)
→ Радиус действия: **~100–500 км**; мгновенный эффект

**Динамический триггер** *(Hill et al., 1993)*
Поверхностные волны Рэлея и Лява
→ Радиус действия: **тысячи км**; распространяются со скоростью ~3–4 км/с

**Вязкоупругая релаксация** *(Pollitz et al., 1998)*
Постсейсмическая деформация нижней коры и мантии
→ Радиус действия: **~1000 км**; задержка **месяцы–годы**

> **Наш метод не устанавливает механизм** — он обнаруживает статистически значимые корреляции. Интерпретация — следующий шаг.

# Ограничения метода

- **Неполнота исторических данных** — для событий до ~1900 CE возможна систематическая недооценка числа землетрясений в слабо задокументированных регионах (Центральная Азия, Африка, Тихий океан)

- **Корреляция ≠ причинность** — обнаружение серий не доказывает физическую связь между событиями

- **Чувствительность к параметрам** — показатель степени `df = 1.6` и b-value `b = 1.0` требуют региональной калибровки; глобальные усреднения могут маскировать локальные аномалии

- **Неопределённость дат** — для событий BCE погрешность дат составляет ±10–100 лет; это существенно влияет на t_ij и значение η

- **Граф плит** — Bird (2003) — статичная модель; не учитывает эволюцию тектоники на исторических масштабах
