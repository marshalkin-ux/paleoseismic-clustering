# План статистического тестирования

**Проект:** paleoseismic-clustering  
**Каталог:** 4267 событий M≥6.5 (4418 CSV-строк), 1900–2026 (исторический хвост до 1900 г.)  
**Цель:** публикация в GRL/BSSA  
**Статус (июнь 2026):** уровни 1–3 выполнены; **итог — null/falsification** (см. `paper/sections/take_home_message.md`)

## Детали уровней тестирования

### Уровень 1 — Пермутационный тест ✅

```
Процедура: случайные перестановки временных меток (n=10 000)
           сохранение координат и магнитуд
Метрика:   среднее log10(η) всех ближайших соседей
Результат:
  • Современный (1973–2026):    p = 0.0001 (1/10 001),  z = −6.17
  • Ранний инструм. (1900–1972): p = 0.007,   z = −2.43
  • Исторический (до 1900):      p = 0.46,    z = −0.74  (мало данных)
Вывод:  Отвергает глобальный пуассоновский нуль по временам — ожидаемо при афтершоках;
        НЕ является тестом гипотезы о мультирегиональных «глобальных сериях»
```

### Уровень 2 — ETAS-валидация ✅

```
Параметры ETAS (MLE на 2041 событии 1973–2026, results/etas_calibration.json):
  μ≈0.103, K≈0.495, α≈0.063, c≈10⁻⁴ сут., p≈1.36

Процедура:
  1. Калибровка ETAS на реальном каталоге (GK + Omori MLE)
  2. Сгенерировать 1000 синтетических ETAS-каталогов (seed=42)
  3. Запустить алгоритм поиска «серий» на каждом
  4. Null distribution: {N_series_ETAS}
  5. p_ETAS = P(N_ETAS ≥ N_observed = 27)

Результат:
  mean=27,0 ± 0,0; max=27; FPR=1000/1000; p_ETAS=1,0
  Multiseed (seeds 42–46, n=100/seed): FPR=1,0 стабилен

Интерпретация:
  p_ETAS=1,0 → N_obs=27 НЕОТЛИЧИМО от ETAS-like локальной кластеризации
  Отрицательный результат по гипотезе избыточной глобальной структуры
  Литературные параметры (μ=0.008) — только для сравнения (mean≈15,4)
```

### Уровень 3 — FDR коррекция Бенджамини–Хохберга ✅

```
Процедура:
  142 окна-кандидата → merge → 47 объединённых эпизодов
  BH на N=47 p-values серий (одна гипотеза на merge-эпизод)

Параметры:
  q = 0.05 (FDR threshold)
  N = 47 (post-hoc; НЕ полное пространство поиска 142×параметры)

Результат: 45/47

Ограничение (критическое):
  FDR на N=47 НЕ корректирует 142 окна × параметры
  45/47 — чувствительность либерального детектора, НЕ discovery claim
```

### Уровень 4 — Анализ чувствительности 🔲

```
Выполнено частично (results/clustering_sensitivity_strict.json):
  min_events=5, min_regions=4 (и жёстче) НЕ снижают N=27 на современном окне
  → либеральность в пространстве поиска (142 окна), не в малых N

Запланировано:
  T_window, α_distance, GK vs ZBZ, полный Jaccard sweep
```

### Уровень 5 — ΔCFS физическая валидация 🔲 FUTURE WORK

```
Для кандидатов: S047, S170, S095 (описательные примеры, не «открытые серии»)

Предварительные тесты Coulomb/динамического стресса НЕ достигли порогов триггеринга
Требует focal mechanisms (GCMT), Coulomb 3.3 / pyCSEP
```
