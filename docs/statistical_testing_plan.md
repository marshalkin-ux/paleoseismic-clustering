## Детали уровней тестирования

### Уровень 1 — Пермутационный тест ✅

```
Процедура: случайные перестановки временных меток (n=10 000)
           сохранение координат и магнитуд
Метрика:   среднее log10(η) всех ближайших соседей
Результат:
  • Современный (1973–2026):    p < 0.0001,  z = −6.17  ★★★★
  • Ранний инструм. (1900–1972): p = 0.007,   z = −2.43  ★★★
  • Исторический (до 1900):      p = 0.46,    z = −0.74  (мало данных)
Вывод:  Отвергаем H₀ для двух из трёх эпох
```

### Уровень 2 — ETAS-валидация 🔲

```
Параметры ETAS (подбираются MLE):
  μ ≈ 0.01, K ≈ 0.08, α ≈ 1.0, c ≈ 0.01, p ≈ 1.1

Процедура:
  1. Подобрать параметры ETAS к реальному каталогу (MLE)
  2. Сгенерировать 100 синтетических ETAS-каталогов
  3. Запустить алгоритм поиска глобальных серий на каждом
  4. Построить null distribution: {N_series_ETAS}
  5. p_ETAS = P(N_ETAS ≥ N_observed = 27)

Интерпретация:
  p_ETAS < 0.01  → результат устойчив к афтершоковой структуре  (публикабельно в GRL)
  0.01–0.05     → погранично; ужесточить пороги (N≥5, ≥4 зоны)
  > 0.05        → заявить ограничения; фокус на методологии

Вычислительные затраты: ~15–40 мин на 8 ядрах
```

### Уровень 3 — FDR коррекция Бенджамини–Хохберга 🔲

```
Формула BH:
  Сортировать p_(1) ≤ p_(2) ≤ … ≤ p_(M)
  k = max{ i : p_(i) ≤ (i/M) × q }
  Отклонить H₀ для i = 1, …, k

Параметры:
  q = 0.05 (FDR threshold)
  M = число кандидатов в серии (ожидается 100–500 при скользящем окне)

Поправка на коррелированные тесты:
  M_eff < M (из-за перекрытия окон)
  Метод Gao et al. (2008): M_eff через собственные числа корр. матрицы

Ожидаемый результат:
  • При текущих p < 0.0001 для 27 серий: все 27 сохранятся
  • Дополнительные кандидаты с p ~ 0.01–0.05: часть исчезнет
```

### Уровень 4 — Анализ чувствительности 🔲

```
Варьируемые параметры:
  T_window:    1–5 лет (±20% от опорного)
  α_distance:  0.8–1.5 (вес тектонического расстояния)
  Декластер.:  Gardner–Knopoff vs Zaliapin–Ben-Zion
  N_min:       3–6 событий
  zones_min:   2–5 зон

Метрика устойчивости:
  Jaccard(S_ref, S_perturb) = |S_ref ∩ S_perturb| / |S_ref ∪ S_perturb|
  Порог: Jaccard ≥ 0.70 → результат устойчив

Итого: ~4×3×2×4×4 = 384 комбинации параметров
       × 1 прогон = ~6 часов на 1 ядре, ~45 мин на 8 ядрах
```

### Уровень 5 — ΔCFS физическая валидация 🔲

```
Для серий: S047 (1905–1910), S170 (2002–2023), S095 (1960–1965)

Методология:
  1. Для каждой пары событий в серии: предыдущее → следующее
  2. Геометрия разрыва: Wells & Coppersmith (1994)
  3. ΔCFS = Δσ_s + μ'·Δσ_n, μ' = 0.4 (King et al. 1994)
  4. Порог: ΔCFS > 0.01 МПа → физически значимый триггер (Stein 1999)

Инструмент: Coulomb 3.3 (Toda et al. 2011) или pyCSEP
Требования: focal mechanisms для каждого события
            (GCMT: ~70% покрытие после 1976)

Ожидаемый результат:
  • Для субдукционных зон: ~60–80% пар имеют ΔCFS > 0.01 МПа
  • Для трансформных: ~40–60%
  • Дальние межрегиональные пары: возможно < 0.01 МПа → роль динамических напряжений
```
