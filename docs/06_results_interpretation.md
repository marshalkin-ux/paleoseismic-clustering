# 06. Интерпретация результатов

## 6.1 Как читать матрицу полноты

Матрица полноты `Mc[region, epoch]` показывает минимальную магнитуду, начиная с которой каталог является **статистически полным** в данном регионе Флинна–Энгдала за данный временной период.

**Что показывают значения:**

- **Mc = 5.0–5.5**: каталог хорошо задокументирован; применимы все статистические методы.
- **Mc = 6.0–6.5**: умеренная полнота; при анализе следует ограничить выборку.
- **Mc = 7.0+**: данные неполны; историческая сейсмичность, высокая неопределённость.

**Рекомендации по интерпретации:**

1. Используйте только события с `magnitude >= Mc[region, epoch]` для данного региона и периода.
2. Сравнивайте матрицы полноты для разных подвыборок — изменение Mc по эпохам отражает развитие сети сейсмических станций.
3. Резкое снижение Mc после 1964 года характерно для всех регионов (введение WWSSN-сети).
4. Внутриплитные зоны (FE regions: 40–60, Евразийская платформа) имеют устойчиво более высокое Mc из-за разреженности станций.

**Пример значений:**

| Регион | 1900–1939 | 1940–1963 | 1964–1975 | 1976–2004 | 2005–2024 |
|--------|-----------|-----------|-----------|-----------|-----------|
| Japan region (FE 19) | 6.5 | 6.0 | 5.5 | 5.0 | 4.5 |
| South America (FE 135) | 7.0 | 6.5 | 6.0 | 5.5 | 5.0 |
| Mid-Indian Ridge (FE 56) | 7.5 | 7.0 | 6.5 | 6.0 | 5.5 |
| East Africa (FE 48) | 7.5 | 7.0 | 6.5 | 6.0 | 5.5 |

## 6.2 Как интерпретировать η-дендрограмму

η-дендрограмма строится методом иерархической кластеризации с метрикой η и алгоритмом single linkage. Она показывает **иерархическую структуру близости** событий в масштабированном пространственно-временном пространстве.

**Элементы дендрограммы:**

- **Листья**: отдельные сейсмические события (подписаны годом и магнитудой).
- **Горизонтальные линии**: слияние кластеров на уровне η.
- **Высота слияния**: значение η на котором два события (или группы) объединяются; **чем ниже — тем теснее связь**.
- **Цвет листьев**: регион Флинна–Энгдала.

**Ключевые наблюдения:**

1. **Компактные кластеры на малых η** (<10^-5): афтершоковые серии, сильно связанные события. Пример: серии ETAS-кластеров вокруг M≥7.5.
2. **Межрегиональные кластеры**: когда листья разных цветов объединяются на умеренных η, это кандидат на глобальную серию.
3. **Изолированные листья** (слияние на больших η): внутриплитные одиночные события, не связанные с другими.
4. **"Вилы" на малой высоте**: форшок-основной толчок-афтершоковая серия.

**Порог η_threshold** на дендрограмме показан горизонтальной пунктирной линией. События, объединившиеся ниже этого порога, считаются связанными.

## 6.3 Что означает p-value в контексте сейсмологии

p-value в данном проекте отвечает на вопрос: **насколько вероятно наблюдать такое или большее число глобальных серий, если временна́я последовательность событий случайна?**

**Интерпретация:**

| p-value | Вывод |
|---------|-------|
| p < 0.01 | Наблюдаемая кластеризация статистически значима на уровне 1%; H0 отвергается. Временная связанность событий выше, чем можно ожидать при случайном порядке. |
| p < 0.05 | Значимость на уровне 5%. Стандартный порог в науке о Земле. |
| p > 0.10 | Нет статистических оснований утверждать, что серии не случайны. |

**Критически важно понимать:**

- **p-value не измеряет физическую причинно-следственную связь** между землетрясениями в разных регионах. Два события могут быть статистически "близки" в η-пространстве без прямого физического взаимодействия.
- **Малое p-value** может быть следствием неоднородности каталога (разная полнота в разных регионах), а не реальной кластеризации.
- Результаты следует интерпретировать совместно с матрицей полноты и геологическим контекстом.

## 6.4 Исторические примеры глобальных серий

В таблице приведены три гипотетических примера, иллюстрирующих применение метода к историческим данным.

### Пример 1: 1906 — серия Сан-Франциско–Вальпараисо

| Параметр | Значение |
|----------|---------|
| Период | Январь–ноябрь 1906 |
| Главные события | M7.9 Сан-Франциско (апрель), M8.2 Вальпараисо (август), M8.0 Эквадор (январь) |
| Регионы FE | 71 (Центральная Калифорния), 134 (Центральное Чили), 137 (Западное побережье Южной Америки) |
| N_связанных | 5 событий с M≥6.5 |
| η_min | ~10^-4 (умеренная связность) |
| p-value | ~0.03 (значимо) |
| Примечание | Каталог неполный (Mc~7.0); оценки предварительны |

**Интерпретация**: 1906 год характеризуется необычно высокой концентрацией крупных землетрясений вдоль Тихоокеанского огненного кольца. Статистическая значимость может частично объясняться неполнотой исторического каталога.

---

### Пример 2: 1960–1964 — серия Чили–Аляска

| Параметр | Значение |
|----------|---------|
| Период | Май 1960 — март 1964 |
| Главные события | M9.5 Чили (май 1960), M9.2 Аляска (март 1964) |
| Сопутствующие события | M8.3 Тихий океан (1963), M8.1 Курилы (1963) |
| Регионы FE | 135 (Чили), 2 (Аляска), 26 (Курилы) |
| N_связанных | 6 событий с M≥7.5 |
| η_min | ~10^-6 (высокая связность) |
| p-value | ~0.004 (высокозначимо) |
| Примечание | Наиболее документированная пара мегаземлетрясений |

**Интерпретация**: Период 1960–1964 гг. включает два крупнейших инструментально зарегистрированных землетрясения. Высокая значимость объясняется исключительной концентрацией M≥8 событий. Физическая связь через глобальную передачу напряжений (stress transfer) обсуждается, но не доказана.

### Пример 3: 2004–2007 — серия Суматра и последующие M8+

| Параметр | Значение |
|----------|---------|
| Период | Декабрь 2004 — январь 2007 |
| Главные события | M9.1 Суматра (декабрь 2004), M8.6 Суматра (март 2005), M8.0 Тонга (май 2006), M8.3 Курилы (ноябрь 2006), M8.1 Курилы (январь 2007) |
| Регионы FE | 154 (Суматра), 63 (Тонга), 26 (Курилы) |
| N_связанных | 8 событий с M≥7.5 |
| η_min | ~10^-7 (очень высокая связность) |
| p-value | ~0.001 (высокозначимо) |
| Примечание | Наиболее полный каталог; интерпретация наиболее надёжна |

**Интерпретация**: Период после Суматры-2004 демонстрирует наивысшую концентрацию M≥8 событий в инструментальный период. Статистическая значимость высокая, однако физическая интерпретация дискуссионна: часть событий может быть связана через динамическое разрушение, часть — через статическое переключение напряжений, часть — быть статистически независимой.

## 6.5 Ограничения метода

### Неравномерность исторических данных

- До 1964 г. каталог существенно **неполон** для M < 7.0. Это систематически занижает число событий в ранних периодах и искусственно уменьшает η (делая события "ближе" из-за разреженности данных).
- Рекомендация: применять анализ только для M ≥ Mc[region, epoch] из матрицы полноты.

### Проблема каузальности vs корреляции

- Метрика η измеряет **статистическую близость** в масштабированном пространстве, но **не физическую причинно-следственную связь**.
- Два события могут демонстрировать низкое η из-за: (а) реальной физической связи через стрессовые поля, (б) общего тектонического контекста (оба на одном тектоническом блоке), (в) случайного совпадения.
- Монте-Карло тест исключает объяснение (в), но не различает (а) и (б).

### Параметрическая чувствительность

- Результаты чувствительны к выбору df (фрактальная размерность) и b-value.
- Рекомендуется проводить анализ чувствительности: варьировать df в диапазоне 1.4–1.8, b в диапазоне 0.8–1.2 и проверять стабильность выводов.

### Граф разломов

- Граф GEM–Bird не включает внутриплитные разломы с малой скоростью скольжения (<0.1 мм/год), что может завышать тектоническое расстояние для умеренных внутриплитных событий.
