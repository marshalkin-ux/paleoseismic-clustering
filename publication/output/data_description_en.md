# Data Description — Global Seismic Series: Statistical Analysis of Spatiotemporal Clustering in M≥6.5 Earthquake Catalogs, 1973–2026 CE

**Author:** Yaroslav Marshalkin  
**Date:** 2026-06-27  
**Repository:** https://github.com/marshalkin-ux/paleoseismic-clustering

## Abstract

We test the hypothesis of physically meaningful **multi-regional global seismic series** in a merged catalog of **4,267 unique M≥6.5 events** (4,418 CSV rows; ~151 NOAA M<6.5 rows retained for provenance, excluded from clustering) using the Baiesi–Paczuski metric η with a **heuristic metric with tectonic hint** (Bird 2003 graph). The detector yields **47 algorithmic candidates** (142 window candidates before merge; 27 modern); **however, ETAS validation (pETAS = 1.0) shows that these candidates are indistinguishable from background activity**. **Primary result — negative (null/falsification):** on the modern window (1973–2026, 2,041 events), the detector finds on average **27.0** candidates in catalog-calibrated ETAS synthetic catalogs, matching the observed count of **27** — the detector is **not specific** to global series. The permutation test (n = 10,000, **p = 0.0001 (1/10,001)**, z = −6.17) rejects a **global temporal Poisson null** — trivial for seismic catalogs; **not** a test of the multi-regional series hypothesis. **98%** of pairs use GC fallback (failed hypothesis). **ΔCFS/dynamic stress — future work only**. Limitations — §5.5.

**Keywords:** global seismicity; seismic series; heuristic metric with tectonic hint; ETAS validation; clustering; Baiesi–Paczuski; Monte Carlo; Flinn–Engdahl

## Conclusion

Catalog-calibrated ETAS shows that the number of multi-regional clusters flagged by our liberal detector does not exceed what is expected from local aftershock activity alone. The hypothesis of physically meaningful global seismic series is therefore not supported. The permutation test rejects only a temporal Poisson null — trivial for earthquake catalogs with aftershocks.
