# Data Description — Global Seismic Series: Statistical Analysis of Spatiotemporal Clustering in M≥6.5 Earthquake Catalogs (1900–2026)

**Author:** Yaroslav Marshalkin  
**Date:** 2026-06-27  
**Repository:** https://github.com/marshalkin-ux/paleoseismic-clustering

## Abstract

We tested the hypothesis of physically meaningful **multi-regional global seismic series** in a merged catalog of **4,267 unique M≥6.5 events** (4,418 CSV rows; ~151 NOAA M<6.5 rows retained for provenance; 1900–2026 with a historical tail before 1900) using the Baiesi–Paczuski metric η and a **heuristic metric with tectonic hint** (Bird 2003 graph). **Primary result — negative (null/falsification):** on the modern window (1973–2026, 2,041 events), catalog-calibrated ETAS without long-range links (>500 km) reproduces **N_obs = 27** (**mean = 27.0**, **p_ETAS = 1.0**, FPR = 1000/1000; multiseed stable across seeds 42–51) — **no excess global structure beyond ETAS-like local clustering**. The permutation test (n = 10,000, **p = 0.0001 (1/10,001)**, z = −6.17) rejects a **global temporal Poisson null** — trivial for seismic catalogs; **not** a test of the multi-regional series hypothesis. The detector yields **47 algorithmic candidates** (142 windows before merge). In **98%** of pairs the metric falls back to great-circle distance (failed hypothesis). **Historical data (pre-1900) are not statistically significant** (p = 0.46). S170, S047 are **descriptive** candidates; ΔCFS/dynamic stress — **future work**.

**Keywords:** global seismicity; seismic series; heuristic metric with tectonic hint; ETAS validation; clustering; Baiesi–Paczuski; Monte Carlo; Flinn–Engdahl

## Conclusion

Catalog-calibrated ETAS shows that the number of multi-regional clusters flagged by our liberal detector does not exceed what is expected from local aftershock activity alone. The hypothesis of physically meaningful global seismic series is therefore not supported. The permutation test rejects only a temporal Poisson null — trivial for earthquake catalogs with aftershocks.
