# Описание данных — Глобальные сейсмические серии: статистический анализ пространственно-временной кластеризации землетрясений M≥6.5 за 1973–2026 гг.

**Автор:** Ярослав Маршалкин  
**Дата:** 2026-06-27  
**Репозиторий:** https://github.com/marshalkin-ux/paleoseismic-clustering

## Аннотация

Проверена гипотеза о физически значимых **мультирегиональных глобальных сериях** в объединённом каталоге **4267 уникальных событий M≥6.5** (4418 CSV-строк; ~151 строка NOAA M<6.5 сохранена для провенанса, исключена из кластеризации) с метрикой Байеси–Пачуски η и **эвристической метрикой с тектонической подсказкой** (граф Bird 2003). Детектор выдаёт **47 алгоритмических кандидатов** (142 окна до merge; 27 на современном окне); **однако ETAS-валидация (p_ETAS=1,0) показывает, что эти кандидаты неотличимы от фоновой активности**. **Главный результат — отрицательный (null/falsification):** на современном окне (1973–2026, 2041 событие) детектор находит в среднем **27,0** кандидатов в ETAS-каталогах с калиброванными параметрами, что совпадает с наблюдаемым числом **27** — детектор **не специфичен** для глобальных серий. Перестановочный тест ($n=10{,}000$, **$p=0{,}0001$ (1/10 001)**, $z=-6{,}17$) отвергает **глобальный пуассоновский нуль по временам** — тривиально для сейсмических каталогов; **не** тест гипотезы о мультирегиональных сериях. **98%** пар — GC-фолбэк (failed hypothesis). **ΔCFS/динамический стресс — future work**. Ограничения — §5.5.
