# Описание данных — Глобальные сейсмические серии: статистический анализ пространственно-временной кластеризации землетрясений M≥6.5 за 1973–2026 гг.

**Автор:** Ярослав Маршалкин  
**Дата:** 2026-06-27  
**Репозиторий:** https://github.com/marshalkin-ux/paleoseismic-clustering

## Аннотация

Проверена гипотеза о физически значимых **мультирегиональных глобальных сериях** в объединённом каталоге **4267 уникальных событий M≥6.5** (4418 CSV-строк; ~151 строка NOAA M<6.5 сохранена для провенанса, исключена из кластеризации) с метрикой Байеси–Пачуски η и **расстоянием по дуге большого круга**. Детектор выдаёт **47 алгоритмических кандидатов** (142 окна до merge; 27 на современном окне); **калиброванная temporal ETAS (p_ETAS=1,0, mean=27,0)** показывает согласие N_obs с null — аномалий временной кластеризации не выявлено. **Главный результат — отрицательный (null):** детектор не специфичен для глобальных серий; пространственная связанность не тестировалась. Перестановочный тест ($n=10{,}000$, **$p=0{,}0001$ (1/10 001)**, $z=-6{,}17$) отвергает **пуассоновские времена** — ожидаемо для каталогов с афтершоками; **не** доказательство глобальных цепочек. **ΔCFS/динамический стресс — future work**. Ограничения — §5.
