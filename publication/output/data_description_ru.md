# Описание данных — Глобальные сейсмические серии: статистический анализ пространственно-временной кластеризации землетрясений M≥6.5 (1900–2026)

**Автор:** Ярослав Маршалкин  
**Дата:** 2026-06-27  
**Репозиторий:** https://github.com/marshalkin-ux/paleoseismic-clustering

## Аннотация

Проверена гипотеза о физически значимых **мультирегиональных глобальных сериях** в объединённом каталоге **4267 уникальных событий M≥6.5** (4418 CSV-строк; ~151 строка NOAA M<6.5 сохранена для провенанса; 1900–2026 с историческим хвостом до 1900 г.) с метрикой Байеси–Пачуски η и тектоническим расстоянием (граф Bird 2003). **Главный результат — отрицательный (null/falsification):** на современном окне (1973–2026, 2041 событие) каталог-откалиброванный ETAS без дальних связей (>500 km) воспроизводит **N_obs=27** (**mean=27,0**, **p_ETAS=1,0**, FPR=1000/1000) — **избыточной глобальной структуры сверх ETAS-подобной локальной кластеризации нет**. Перестановочный тест ($n=10{,}000$, **$p=0{,}0001$ (1/10 001)**, $z=-6{,}17$) отвергает **глобальный пуассоновский нуль по временам** — ожидаемо при афтершоках; **не** тест гипотезы о мультирегиональных сериях. Детектор выдаёт **47 алгоритмических кандидатов** (142 окна до merge); FDR 45/47 при $q=0{,}05$ **не** корректирует полное пространство поиска. Кандидаты **неотличимы** от ETAS-null и артефактов либерального exploratory search. **Исторические данные (до 1900) статистически не значимы** ($p=0{,}46$). S170, S047 и др. — **описательные** примеры крупных кандидатов, не «открытые серии»; ΔCFS/динамический стресс — **только future work**.
